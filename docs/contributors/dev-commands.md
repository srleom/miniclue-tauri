# Development Commands

## Local Development

```bash
bun run dev
bun run dev:fe
```

## Build

```bash
bun run build
bun run build:fe
bun run preview
```

## Type Generation

```bash
bun run gen:bindings
```

## Frontend Checks

```bash
bun run test:ts
bun run lint
bun run lint:fix
bun run format
bun run format:check
bun run fix
bun run check
```

## Rust Checks

```bash
bun run rust:fmt
bun run rust:lint
bun run rust:test
bun run rust:check
```

## Full Validation

```bash
bun run check:all
```
